# fileLock是一个通过异或运算进行加密的cli软件
## 可以将应用程序添加到环境变量中或者添加到Windows文件夹下
## 开发的初衷是为了以一种简单快捷的方式加密作者的H资源[手动滑稽]

## Windows环境下，下载后编译完成后双击bat脚本即可完成配置
# 使用说明
## 在命令行输入
## file_lock.exe [password] [read_file] [write_file]
## file_lock.exe [密码] [被加密文件] [生成文件]
